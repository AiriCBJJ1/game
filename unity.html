Unity 製作說明

從構思到實作，帶你了解我們如何用 Unity 打造這款遊戲

開發流程簡介

本遊戲由 Unity 引擎打造，採用 2D 與 3D 混合製作，並結合聊天機器人與影像辨識系統，實現互動性與沉浸感兼具的遊戲體驗。

使用工具與插件

  • Unity 2022 LTS
  • Standalone File Browser（圖片檔案選取）
  • VRoid Studio（角色建模）
  • Mixamo（動畫綁定與動作捕捉）
  • PixAI、Suno（圖像與音樂）

重要系統實作

血條系統： 透過 Singleton 模式搭配 DontDestroyOnLoad 實作，確保切換場景時血條資訊不會遺失。

圖片選取功能： 使用 SFB 套件實現 Windows 平台開啟檔案總管，只允許選擇 .jpg 圖片進行辨識。
